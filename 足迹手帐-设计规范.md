# 足迹手帐 设计规范 v1.0

> 核心隐喻：**手帐本** — 一本正在被书写的个人记录册
>
> 所有设计决策从「手帐」的物理体验出发：翻开、书写、收藏、回顾

---

## 一、色彩系统

### 1.1 品牌色（手帐本底色）

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#FAF7F2` | 暖米白，模拟手帐纸张 |
| 卡片底色 | `#FFFDF9` | 奶白，比背景略亮，有层叠感 |
| 文字主色 | `#3D3226` | 深棕，模拟墨水色 |
| 文字辅色 | `#8B7E6E` | 灰棕，注释/辅助信息 |
| 分割线 | `#E8E0D5` | 淡棕，模拟铅笔线 |

### 1.2 6 种地点主题渐变（保留并优化）

| 主题 | 渐变 | 首字底色 | 适用 |
|------|------|----------|------|
| 樱花 | `#FF9A9E → #FAD0C4` | `#FF8A80` | 美食、甜点 |
| 森林 | `#A8E6CF → #DCEDC1` | `#81C784` | 公园、户外 |
| 海洋 | `#89CFF0 → #B5EAD7` | `#64B5F6` | 水边、旅行 |
| 日落 | `#FFD3A5 → #FD6585` | `#FFB74D` | 城市、夜景 |
| 紫罗兰 | `#C9B1FF → #FFACE0` | `#BA68C8` | 文艺、书店 |
| 暖阳 | `#F8E2A6 → #F4A460` | `#FFCC80` | 咖啡、日常 |

> 分配规则：取地点名称首字 Unicode 码点 `% 6` 映射到 6 个主题。
> 所有渐变为 CSS `linear-gradient`，无图片依赖。

### 1.3 功能色

| 用途 | 色值 |
|------|------|
| 主按钮 | `#E8775B`（陶土红，温暖不刺眼） |
| 次要按钮 | `#F5E6D8`（淡奶油） |
| 成功/确认 | `#7BC67E` |
| 提示/警告 | `#F4C97E` |
| 错误/删除 | `#E88B8B` |
| 新消息 | `#FF6B6B` |

## 二、字体系统

### 2.1 字号层级

| 层级 | 大小 | 用途 | 字重 |
|------|------|------|------|
| H1 | 36rpx | 页面大标题 | 600 |
| H2 | 32rpx | 卡片标题 | 600 |
| H3 | 28rpx | 区块标题 | 500 |
| Body | 28rpx | 正文 | 400 |
| Caption | 24rpx | 辅助信息、时间戳 | 400 |
| Small | 20rpx | 标签、标注 | 400 |

### 2.2 字体风格

- **手帐标题：** 可考虑使用有手写感的字体（如站酷快乐体），用于「我的」页面签名和特别标注
- **正文：** 使用系统默认字体（苹方 / 思源），保证可读性
- **数字：** 统计数据使用略粗的字重，加强对比

## 四、阴影系统

模拟纸张层叠感，用柔和多层阴影：

| 层级 | 阴影 |
|------|------|
| 卡片（默认） | `0 2rpx 12rpx rgba(61, 50, 38, 0.06)` |
| 卡片（悬浮） | `0 4rpx 20rpx rgba(61, 50, 38, 0.10)` |
| 弹窗 / 浮层 | `0 8rpx 32rpx rgba(61, 50, 38, 0.14)` |
| 贴纸 | `0 1rpx 4rpx rgba(61, 50, 38, 0.08)` + 轻微旋转（±3deg） |

> 所有阴影使用手帐纸张的棕色调，不用纯黑阴影。

## 七、微交互规范

### 7.1 转场动画

- 页面切换：`opacity 0.2s` + `translateY(8rpx → 0)`，柔和淡入
- 弹窗出现：`scale(0.95 → 1)` + `opacity 0 → 1`，200ms
- 卡片列表：`staggered animation`，每张卡片延迟 50ms

### 7.2 反馈动画

- 收藏成功：卡片轻微弹跳 + 一颗小星星飘出（纯 CSS）
- 打卡完成：打卡计数 +1 的数字跳动动画
- 贴纸贴上：贴纸从贴纸本「飞入」卡片角标的动画（200ms）
- 贴纸解锁：灰色剪影 → 彩色 + 微微弹跳

### 7.3 手势反馈

- 卡片点击：轻微缩小（`scale(0.98)`）+ 阴影加深
- 筛选词切换：平滑的指示器滑动（`transition: left 0.3s ease`）

## 八、空状态设计

| 页面 | 空状态 | 引导文案 |
|------|--------|----------|
| 地点页 | 一本合上的手帐本插画 + 柔和阴影 | 「翻开手帐，记录你去过的地方」 |
| 旅程页 | 一张空地图 + 轻描淡写的虚线 | 「开始收藏地点，你的足迹会出现在这里」 |
| 路线页 | 一条虚线路径 | 「创建第一条路线，记录你的旅程」 |
| 贴纸本 | - | （无空状态，基础贴纸全量开放） |

> 所有空状态插图用纯 CSS / SVG 绘制，不加包体积。

## 十、设计原则速查

| 做 | 不做 |
|----|------|
| 柔和圆角 | 直角 / 尖锐边角 |
| 纸张质感阴影 | 系统默认硬阴影 |
| 墨水色文字 | 纯黑 `#000` 文字 |
| 暖色调氛围 | 冷色调为主 |
| 留白呼吸 | 内容拥挤 |
| 手帐元素点缀 | 过度装饰 |
| 纯 CSS 实现 | 图片依赖 |
| 温婉微动效 | 大范围动画 |
| 用户语言文案 | 技术术语 / 抽象概念 |
