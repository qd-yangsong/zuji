需要配置的真实凭证共 **5 项**，按优先级排列如下：

---

## 1. 微信小程序 AppID

**用途**：小程序的唯一标识，登录、发布、API 调用都依赖它

**在哪里获取**：
1. 访问 https://mp.weixin.qq.com/
2. 注册一个小程序账号（选择"小程序"类型）
3. 注册完成后，进入 **开发管理 -> 开发设置**
4. 顶部即可看到 **AppID（小程序ID）**，格式如 `wx1a2b3c4d5e6f7g8h`

**填在哪里**：
- 文件：[project.config.json](file:///Users/yangsong/个人产品设计/2026_6_17/apps/miniapp/project.config.json) 第 5 行
- 把 `"appid": "touristappid"` 改为你的真实 AppID

---

## 2. 微信小程序 AppSecret

**用途**：后端用来调用微信登录接口（`code2Session`），用 AppID + AppSecret 换取用户的 openid

**在哪里获取**：
1. 同样在 https://mp.weixin.qq.com/ -> **开发管理 -> 开发设置**
2. 找到 **AppSecret（小程序密钥）**，点击"生成"或"重置"
3. 用管理员微信扫码确认后会显示一次，**务必立即保存**（关闭后无法再查看）

**填在哪里**：

- 文件：[apps/server/.env](file:///Users/yangsong/个人产品设计/2026_6_17/apps/server/.env.example)（从 `.env.example` 复制为 `.env`）
- `WX_APPID=你的AppID`
- `WX_SECRET=你的AppSecret`

---

## 3. 腾讯云 COS（对象存储）密钥

**用途**：存储用户上传的封面图、打卡照片、反馈截图等图片

**在哪里获取**：
1. 访问 https://console.cloud.tencent.com/
2. 开通 **对象存储 COS** 服务
3. 创建一个存储桶（Bucket），地域建议选离你服务器最近的（如 `ap-shanghai`）
4. 访问 https://console.cloud.tencent.com/cam/capi
5. 点击"新建密钥"，获得 **SecretId** 和 **SecretKey**

**填在哪里**：
- 文件：[apps/server/.env](file:///Users/yangsong/个人产品设计/2026_6_17/apps/server/.env.example)
- `COS_SECRET_ID=你的SecretId`
- `COS_SECRET_KEY=你的SecretKey`
- `COS_BUCKET=你的存储桶名`（如 `zuji-1234567890`）
- `COS_REGION=ap-shanghai`（与你创建存储桶时选的地域一致）

---

## 4. JWT Secret

**用途**：后端签发和验证用户登录 Token 的密钥

**在哪里获取**：
- 不需要从第三方获取，自己生成一个随机字符串即可
- 在终端执行：`openssl rand -base64 32`

**填在哪里**：
- 文件：[apps/server/.env](file:///Users/yangsong/个人产品设计/2026_6_17/apps/server/.env.example)
- `JWT_SECRET=生成的随机字符串`

---

## 5. 小程序后端 API 域名

**用途**：小程序前端请求后端 API 的地址。你之前提到 `track.highnix.cn` 可以用于此用途。

**在哪里配置**：
1. 文件：[apps/miniapp/config/prod.ts](file:///Users/yangsong/个人产品设计/2026_6_17/apps/miniapp/config/prod.ts) 第 8 行
   - 把 `https://你的域名.com/api` 改为 `https://track.highnix.cn/api`
2. 微信后台：https://mp.weixin.qq.com/ -> **开发管理 -> 开发设置 -> 服务器域名**
   - 在 **request 合法域名** 中添加 `https://track.highnix.cn`
   - 在 **uploadFile 合法域名** 中添加你的 COS 域名（如 `https://zuji-1234567890.cos.ap-shanghai.myqcloud.com`）

---

## 汇总清单

| 序号 | 凭证                     | 获取地址                               | 填入文件                                           |
| ---- | ------------------------ | -------------------------------------- | -------------------------------------------------- |
| 1    | AppID                    | 微信公众平台 -> 开发设置               | `project.config.json`                              |
| 2    | AppSecret                | 微信公众平台 -> 开发设置（扫码生成）   | `apps/server/.env`                                 |
| 3    | COS SecretId / SecretKey | 腾讯云控制台 -> 访问管理 -> API密钥    | `apps/server/.env`                                 |
| 4    | COS Bucket / Region      | 腾讯云控制台 -> 对象存储（创建存储桶） | `apps/server/.env`                                 |
| 5    | JWT Secret               | 终端执行 `openssl rand -base64 32`     | `apps/server/.env`                                 |
| 6    | API 域名                 | 你已有的 `track.highnix.cn`            | `apps/miniapp/config/prod.ts` + 微信后台服务器域名 |

---

**额外注意**：微信公众平台注册小程序时需要完成**主体认证**（个人或企业），认证后才能正式发布。未认证的开发阶段可以用微信开发者工具的"不校验合法域名"选项进行本地调试。
