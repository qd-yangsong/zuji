# 足迹手帐 · 单页面网站开发实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将已定稿的高保真原型 `prototype/landing.html` 整理为一个可独立部署、路径规范、素材自包含的单页面网站。

**Architecture:** 采用纯静态网站结构，入口为 `web/landing/index.html`，样式拆分为独立 CSS 文件，图片素材复制到 `web/landing/assets/` 目录，避免跨目录引用。保留 Tailwind CDN 以快速实现样式，后续如需优化体积可迁移到构建后的 CSS。

**Tech Stack:** HTML5、Tailwind CSS（CDN）、原生 CSS、原生 JavaScript。

---

## 文件结构

```
web/landing/
├── index.html          # 单页面入口（从 prototype/landing.html 迁移并规范化）
├── styles.css          # 自定义 CSS：动画、颗粒纹理、特殊字体样式
└── assets/             # 网站自包含素材
    ├── logo/
    │   ├── thumb-96.png
    │   └── logo-400.png
    ├── theme-illust/

    │   ├── park.png
    │   ├── gather.png
    │   └── exhibit.png
    └── empty-state/
        └── journey.png
```

## Task 2: 创建独立样式文件

```css
  }

/* 章节小标签 */

  letter-spacing: 0.25em;
  font-size: 0.75rem;
  text-transform: uppercase;
  font-weight: 500;
  opacity: 0.7;

  .section-label {
    font-size: 0.875rem;

/* 移动端菜单默认隐藏 */
.mobile-menu {
  display: none;
}

.mobile-menu.is-open {
  display: flex;
}
```
